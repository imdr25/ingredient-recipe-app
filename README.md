# Ingredient-Based Recipe App (MVP)

## Overview
A simple cooking app where users list the ingredients they have and receive recipe suggestions.

## MVP Scope
- User types ingredients they have
- User selects cuisine, diet, and difficulty
- App shows 5 recipe options
- Simple, Tasty-style experience

## Goal
Start with the most basic version to validate the idea before adding advanced features.
## Future Features
- Nutrition information per serving
- Fridge photo upload to detect ingredients
- Smarter AI filtering
